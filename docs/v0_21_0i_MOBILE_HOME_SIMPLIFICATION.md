# v0.21.0i — Mobile home simplification

## Objetivo

Simplificar la portada pública, especialmente en móvil, para que el usuario entienda el juego y empiece a jugar en pocos segundos.

## Cambios

- Portada móvil más compacta tipo app.
- Badge superior reducido con versión y estado beta.
- Texto principal más directo.
- Dificultad visible y compacta.
- CTA principal: Nueva partida.
- CTA secundario: Modo carrera.
- Continuar queda desactivado si no hay guardado.
- Compartir juego pasa a enlace discreto.
- “¿Cómo se juega?” queda plegado.
- Bloque de apoyo se mantiene abajo.
- Footer móvil con versión y feedback.

## No se toca

- Simulación.
- Rating efectivo de carrera.
- Ascensos y descensos.
- Calendario dinámico.
- Copa.
- Balance de partida rápida.
