# v0.13.3 — Década 1970/71-1979/80

Añade 10 plantillas jugables de la década de los 70.

## Criterio

- 1970/71-1973/74: continuidad del ciclo Iribar, Rojo, Uriarte.
- 1972/73: temporada reforzada por Copa.
- 1976/77: bloque fuerte por final europea y temporada histórica.
- 1978/79-1979/80: transición hacia el bloque de los 80.

## Nota

Base jugable provisional. Los ratings están calibrados por contexto, peso histórico y lógica jugable. La fase posterior debe refinar con estadísticas oficiales cuando existan.

## Uso

```powershell
cd "D:\Proyectos\Futbol11"
powershell -ExecutionPolicy Bypass -File .\apply-v0-13-3-1970s-squads.ps1
npm.cmd run build
npm.cmd run dev
```
