# v0.13.6 — Década 2000/01-2009/10

Añade las plantillas faltantes de la década de los 2000.

## Nota importante

La temporada 2008/09 ya existía en el proyecto, así que este parche no la duplica.

## Temporadas añadidas

```text
2000/01
2001/02
2002/03
2003/04
2004/05
2005/06
2006/07
2007/08
2009/10
```

## Criterio

- 2000/01: continuidad del bloque Etxeberria-Urzaiz-Yeste.
- 2001/02-2002/03: ciclo Heynckes.
- 2003/04-2004/05: primer ciclo Valverde.
- 2005/06-2006/07: temporadas de crisis, ratings controlados.
- 2007/08-2009/10: ciclo Caparrós, Llorente, Javi Martínez, Iraola, Amorebieta, Susaeta y Muniain.
- 2008/09: ya existente por final de Copa.

## Uso

```powershell
cd "D:\Proyectos\⚽ Futbol11"
powershell -ExecutionPolicy Bypass -File .\apply-v0-13-6-2000s-squads.ps1

npm.cmd run dev
```
