# v0.14.6 — Modo Fácil con rangos + Modo Leyenda

## Cambios aplicados

- El modo accesible se muestra ahora como **Modo Fácil**.
- El modo equilibrado se muestra como **Modo Normal** y mantiene la lógica interna ya existente.
- El antiguo modo difícil queda visible como **Modo Leyenda**, sin cambiar su comportamiento interno.
- Se añade selector de rango de temporadas solo cuando está activo el **Modo Fácil**.

## Rangos disponibles en Modo Fácil

- Todas las temporadas: 1928/29–2025/26.
- Athletic clásico: 1928/29–1959/60.
- Athletic de transición: 1960/61–1989/90.
- Athletic moderno: 1990/91–2009/10.
- Athletic reciente: 2010/11–2025/26.

## Nota técnica

No se han cambiado ratings, simulación, formación, reglas de posición, lógica de Liga ni Copa.
El filtro de temporadas se aplica únicamente al generar o regenerar rondas del draft en el Modo Fácil.
