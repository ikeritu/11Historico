# v0.18.7 — Public Copy and Feedback Form

## Objetivo

Limpiar textos públicos de la beta online y sustituir el feedback por un formulario sencillo.

- Versión visible actualizada a `v0.18.7`.
- Eliminada la referencia a `prueba externa local` en la portada.
- Texto público actualizado: `Liga, Copa, draft y UX móvil validadas. Versión online abierta para pruebas y feedback.`
- El botón `Enviar feedback` apunta ahora a Google Forms.
- Se evita GitHub Issues como canal principal para no exigir cuenta técnica a los usuarios.
