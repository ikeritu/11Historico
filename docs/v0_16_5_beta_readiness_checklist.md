# Checklist beta privada — Futbol11 v0.16.5

## Flujo principal

- [ ] Nueva partida desde cero.
- [ ] Continuar partida guardada.
- [ ] Modo Fácil con cada rango de temporadas.
- [ ] Modo Normal.
- [ ] Modo Leyenda.
- [ ] Draft completo de 11 jugadores.
- [ ] Resortear ronda.
- [ ] Elegir entrenador.
- [ ] Revisar resumen del equipo.
- [ ] Simular con botón `Simular`.
- [ ] Parar simulación.
- [ ] Saltar hasta próximo evento.
- [ ] Llegar a pantalla final.
- [ ] Nueva partida desde pantalla final.

## Responsive

- [ ] Portada en móvil.
- [ ] Selector de dificultad en móvil.
- [ ] Selector de formación en móvil.
- [ ] Draft en móvil.
- [ ] Resumen del equipo en móvil.
- [ ] Simulación en móvil.
- [ ] Pantalla final en móvil.

## Textos

- [ ] Tildes y caracteres especiales.
- [ ] Nombres históricos raros.
- [ ] Botones claros.
- [ ] Mensajes de error comprensibles.
- [ ] No aparecen textos de auditoría dentro del juego.

## Auditorías

- [ ] `npm.cmd run audit:balance:100`.
- [ ] `npm.cmd run audit:copa:100`.

## Criterio de cierre

La beta privada puede avanzar cuando una persona externa pueda jugar una partida completa sin explicación adicional y sin bloqueo crítico.
