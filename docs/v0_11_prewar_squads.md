# Futbol11 v0.11 — Bloque histórico 1928/29 - 1935/36

## Qué añade

Añade las primeras 8 temporadas históricas del Athletic al draft jugable:

- 1928/29
- 1929/30
- 1930/31
- 1931/32
- 1932/33
- 1933/34
- 1934/35
- 1935/36

## Criterio

- Plantillas y categorías de posición basadas en listados oficiales del Athletic Club cuando están disponibles.
- Ratings de gameplay estimados: no son estadística oficial.
- Minutos, goles y asistencias individuales quedan pendientes de verificación fina.
- Se añaden alias/canonicalPlayerId para el bloque pre-Guerra Civil.

## Entrenadores incluidos

- Máximo Royo — 1928/29
- Frederick Pentland — 1929/30 a 1932/33
- Patricio Caicedo — 1933/34 y 1934/35
- William Garbutt — 1935/36

## Después de aplicar

```powershell
npm.cmd run build
npm.cmd run dev
```

## Siguiente bloque recomendado

1939/40 - 1949/50, con especial atención a la aparición del bloque Iriondo / Venancio / Zarra / Panizo / Gainza.
