# v0.13.1 — Década 1950/51-1959/60

## Objetivo

Añadir la década completa de los 50 como bloque jugable.

Temporadas añadidas:

```text
1950/51
1951/52
1952/53
1953/54
1954/55
1955/56
1956/57
1957/58
1958/59
1959/60
```

## Criterio

Esta década se calibra con más peso histórico que la década de posguerra inicial.

Puntos clave:

```text
1950/51-1954/55: transición y consolidación
1955/56: pico de rendimiento, equipo campeón de Liga
1956/57: bloque fuerte con contexto europeo
1957/58: bloque copero fuerte
1958/59-1959/60: cierre de ciclo y transición
```

## Regla de ratings

Siguiendo la regla del proyecto:

```text
Si hay estadísticas oficiales o fiables, deben pesar mucho en el rating.
Las temporadas con menos datos se normalizan a partir de temporadas con datos.
```

Este parche deja una base jugable con `dataConfidence` media y ratings calibrados por:

```text
contexto competitivo
titulos
peso historico del jugador
rol dentro del equipo
logica jugable
```

La fase posterior debe refinar con datos oficiales por jugador.

## QA recomendado

1. Build.
2. Comprobar que aparecen temporadas 1950/51-1959/60 en el sorteo.
3. Probar una temporada fuerte: 1955/56.
4. Probar una temporada de transición: 1950/51 o 1959/60.
5. Verificar que se puede completar once.
6. Pasar a entrenador.
7. Simular temporada completa.
