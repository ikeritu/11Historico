# v0.22.2 - Career reward flow fix

Ajuste de flujo de recompensas de modo carrera.

## Cambios

- Se elimina `Volver al inicio` de la pantalla de final de temporada para reducir errores de salida accidental.
- Recompensa sin título: cambiar jugador o cambiar entrenador.
- Recompensa con título: jugador + alineación o entrenador +0.5 media.
- La Supercopa solo desbloquea premio especial si además hay clasificación europea.
- El cambio de alineación se muestra después de elegir el jugador que sale.
- El entrenador +0.5 permite mantener el entrenador actual o elegir uno nuevo.
- Cancelar una rama de recompensa ya iniciada avanza a la siguiente temporada conservando el estado actual.

## No toca

- Balance Liga/Copa.
- Rating efectivo de carrera.
- Supercopa.
- Ascensos/descensos.
- Calendario dinámico.
